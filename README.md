# 🕵️ AI Writing Style Detector

An AI-powered web app that analyzes any text and tells you:
- 🤖 Is it written by **AI or a Human**?
- 🎭 What **emotion and tone** does it carry?
- 🏷️ What **writing style** is it? (Academic, Corporate, Creative, Casual)

Built with Python, Streamlit, and HuggingFace Transformers.

## ✨ Features

- **AI vs Human Detection** — Detects if text was generated by AI or written by a human, with confidence score and key signals
- **Emotion & Tone Analysis** — Identifies 7 emotions (joy, sadness, anger, fear, surprise, disgust, neutral) with a full breakdown
- **Writing Style Classification** — Classifies writing into Academic, Corporate, Creative, or Casual with scores for each
- **Sample Texts** — Built-in sample texts to test the app instantly
- **Clean Dashboard** — Simple and easy to understand results

## 💡 How It Works


User pastes text
       ↓
AI Detection Model  →  Is it AI or Human written?
Emotion Model       →  What emotion does it carry?
Style Rules         →  What writing style is it?
       ↓
Results displayed on dashboard
